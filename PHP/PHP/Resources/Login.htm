Introduction

This application has been built specially for the people's health pharmacy. It is designed to handle instore transactions and to keep an accurate record of all products available in store.

It can also generate reports that provide insight into sales across the past week and the past month.

This documentation explains how to use the software and all the tools available.

For any other questions or queries, please contact Murcs Developement

Login Page

  1. Enter your user id.
    If you're uncertain as to what this is, see your supervising manager. It is important the you log in with YOUR OWN id and not someone elses so that sales can be accurately recorded.
  2. Enter your password.
    You should have set this when you were first employeed. If you are unsure as to what it is or need to change it for another reason, see your supervising manager.
  3. Press the Login button

Details can be removed and the login process restarted by pressing the clear button
