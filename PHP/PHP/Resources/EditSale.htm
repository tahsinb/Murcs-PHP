Edit Sale

From here past sales can be edited. This is helpful if sales were made incorrectly or if refunds need to be made.

To select a sale, find it in the displayed list of sales, enter the Sale ID in the available text area and 'Edit'

Once the 'Edit' has been pressed, it will show only the selected sale and text areas will display down the bottom in which you can edit the details of the sale

Once you are happy with the details, select 'Confirm', you will be returned to the Edit sales page and the updated details will be saved.
