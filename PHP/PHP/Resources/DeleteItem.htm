Delete Product

Products can be deleted when the store no longer stocks them.

To delete an item, enter its ID in the provided textfield and select the 'Remove' button

Once a product has been deleted from the database it cannot be retrieved. If it's removed acccidentally or needs to be added for another reason, it must be added again using the Add Product function
